# Screenpipe Sidecar Binary

Place the Screenpipe binary here for Tauri to bundle it into the .dmg.

Expected filename: screenpipe-aarch64-apple-darwin (for Apple Silicon Mac)

Download from: https://screenpi.pe/ or build from https://github.com/mediar-ai/screenpipe

If this binary is not present, Ottie will still work but without built-in screen sensing.
Users can install Screenpipe separately: npx screenpipe@latest record
